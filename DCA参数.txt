## 🛠️ 2. 如何用 REST API 下载 Binance 交易对列表

### 📌 Spot 现货交易对

你可以调用官方 Spot REST API：

```
GET https://api.binance.com/api/v3/exchangeInfo
```

这个返回包括所有 spot 交易对及状态，例如：

```json
{
  "symbols": [
     {"symbol": "BTCUSDT", "status": "TRADING", ...},
     {"symbol": "DOGEUSDT", "status": "TRADING", ...},
     ...
  ]
}
```

要过滤出所有 USDT 交易对，可以在拿到返回后筛选：

```python
usdt_pairs = [s["symbol"] for s in data["symbols"] if s["quoteAsset"] == "USDT"]
```

📌 Spot API 文档说明见币安官方。([Binance][1])

---

### 📌 Futures 永续合约交易对

币安的合约交易对需要调用 Futures API：

```
GET https://fapi.binance.com/fapi/v1/exchangeInfo
```

或者：

```
GET https://dapi.binance.com/dapi/v1/exchangeInfo  （USDⓈ-M 永续合约）
```

这会返回所有期货合约符号列表，包括永续合约。可用于判断像 **ACTUSDT**、**PEOPLEUSDT** 等是否存在。

## 📌 实时在 API 验证交易对是否有效

可以用 `/api/v3/exchangeInfo?symbol=XXXUSDT` 检查 Spot 是否有效，例如：

```
GET https://api.binance.com/api/v3/exchangeInfo?symbol=DOGEUSDT
```

如果返回错误就是不存在。
