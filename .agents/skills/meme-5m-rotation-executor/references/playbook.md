# 5m 短线 · MEME轮动 · 网格+趋势 的实战执行框架

这不是理论，是可以每天照着执行的操作系统结构。

## 1) 交易前准备

### 扫描 MEME 池（只选 3-5 个活跃币）

筛选条件：

- 24h 成交量排名前 20
- 5m 成交量明显放大
- ATR(5m) 足够大（有波动才有利润）
- 非明显出货形态

原则：

- 不碰冷币，不提前埋伏，只做正在动的币。

## 2) 市场状态判断（每 30 分钟）

### A. 震荡模式

- EMA9 与 EMA21 缠绕
- 无连续放量
- 多次上下影线

执行：主用网格套利。

### B. 启动趋势模式

- EMA9 > EMA21 且角度明显
- 连续 2-3 根放量阳线
- 突破前高结构

执行：主用趋势盈利。

### C. 情绪极端模式

- 3 根以上超大阳线
- RSI > 80
- 社群 FOMO 爆发

执行：快进快出 / 减仓。

## 3) 入场触发系统

### 网格套利执行逻辑

触发条件：

- 明确震荡区间
- ATR 稳定
- 上下边界确认

参数：

- 网格间距 = ATR × 0.8
- 8-15 格
- 单格仓位 1-2%

止损机制：

- 跌破区间下边界 -> 全部暂停
- 突破区间上沿放量 -> 转趋势模式

目标：高频小利润叠加。

### 趋势单执行逻辑

启动信号必须满足：

- 放量突破前高
- 回踩不破 EMA9
- 成交量不衰减

入场：

- 突破回踩确认时进
- 或第二次放量时进

止损：

- 跌破 EMA21
- 或放量滞涨

止盈：

- 分批止盈 30%
- 其余移动止盈（低点上移）

原则：让利润奔跑，而不是一次全走。

## 4) 持仓管理逻辑

- 不同时持有超过 2 个 MEME（防止情绪共振暴跌）
- 单币最大仓位 <= 20%
- 连续 2 单止损 -> 降低仓位 50%
- 连续盈利 3 单 -> 保持仓位不扩大（防止自信膨胀）

## 5) MEME 轮动切换机制

观察：

- 龙头涨幅超过 15%
- 龙头缩量横盘
- 第二梯队突然放量

执行：

- 龙头减仓
- 切换至补涨币
- 不追高，只做启动阶段

口诀：

- 龙头爆发 -> 关注补涨
- 龙头滞涨 -> 减仓转移
- 龙头放量下跌 -> 全市场降风险

## 6) 风控与暂停机制（极端行情）

插针暴跌：

- 不抄底
- 等二次确认
- 仓位减半

急拉 FOMO：

- 不追第三根大阳
- 等回踩再进

全市场恐慌：

- 直接空仓观察 30 分钟

## 7) 每日风控系统

- 单日最大回撤 5% -> 强制停止
- 连续 3 天亏损 -> 降低仓位模式
- 每周至少 1 天不交易（避免疲劳）

## 8) 每日复盘优化流程

每日复盘四问：

1. 哪些单是按系统执行？
2. 哪些单是情绪交易？
3. 哪种形态胜率最高？
4. 哪种入场最容易被收割？

持续优化：

- 网格间距
- 趋势确认标准
- 轮动切换速度

## 整体执行哲学

在 5m MEME 市场中：

- 网格负责稳定现金流
- 趋势负责拉高收益曲线
- 轮动负责提高资金效率
- 风控负责长期存活

真正赚钱的不是预测方向，而是：

- 在震荡时赚钱
- 在趋势时吃肉
- 在混乱时保命
