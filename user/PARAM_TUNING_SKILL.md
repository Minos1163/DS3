# PARAM_TUNING_SKILL.md

描述
- 该技能文件用于在调试和优化参数时提供标准化的参数调优方法。将其放入用户目录后，AI 可以在需要进行参数调试时读取并执行其中定义的流程与约束。

安装指引
- 将本文件放置于你的工作目录下的 `user/` 文件夹中，例如：`D:/AIDCA/AI2/user/PARAM_TUNING_SKILL.md`。
- 在需要进行参数调试时，调用本技能内容中给出的步骤和约束作为参考执行。

调用方式
- 通过现有技能接口执行参数调优：优先使用 FUND-FLOW 调参技能（fund-flow-param-tuning-cn），若遇到跨域或组合调参需求，则结合 PARAM_TUNING_SKILL.md 中的流程。
- 调用时提供当前系统状态、目标、约束以及希望达到的指标；技能将输出一个可执行的调参计划和推荐的参数值。

调试流程示例（简化）
- 收集输入：当前参数集、目标指标、限制条件（如最大回撤、波动率、吞吐量等）。
- 设定优化目标：如最大化收益/达到某个单位时间的 Sharpe 比率、降低最大回撤等。 
- 运行调参循环：采用网格/随机/贝叶斯等策略，迭代评估并收敛至目标阈值。
- 验证：在沙箱/回测环境中验证新参数的稳定性，记录日志。

示例任务片段
- 目标：将最大回撤控制在 5% 以下，同时在 90 天内实现收益率 > 8%。
- 初始参数：alpha=0.1, beta=0.5, risk_budget=0.2, turnover=0.8
- 期望输出：推荐的新参数集、对应的 Sharpe、最大回撤、夏普比率等。

注意事项
- 不要直接执行带有风险的真实资金操作，优先在仿真/回测环境中验证。
- 参数之间的耦合关系需要记录，避免单变量调参导致其他指标恶化。
